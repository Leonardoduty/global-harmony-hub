You are seeing this page because the folder is being served as plain static files. That mode only lists files unless the server picks up this index.html.

This app is React + TanStack Start and must be run with the Vite dev server (or deployed to Cloudflare Workers), not by opening the directory directly.

Run the real app locally

  1. In a terminal, go to this project folder (where package.json is).
  2. Install once: npm install --legacy-peer-deps
  3. Start and open the browser: npm run dev:open

Or run npm run dev and open the URL shown in the terminal (often http://localhost:8080).
